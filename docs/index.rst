ATPP - Active Thermography Test Post-Processing Documentation
==============================================================

Welcome to the documentation for **ATPP (Active Thermography Test Post-Processing)**, a comprehensive tool designed to help users analyze and process thermography data with ease.

This documentation provides an in-depth guide on how to install, use, and contribute to ATPP, along with detailed information on all the available modules and functions.

Overview
--------

ATPP provides robust post-processing for thermography data, offering features such as noise reduction, thermal visualization, and data filtering. Whether you are a researcher, engineer, or thermography enthusiast, ATPP can streamline your workflow and help you obtain valuable insights from your thermal data.
